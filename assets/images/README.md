# Structure des images

## Dossier `profile/`
- `profile.jpg` - Photo de profil pour la page d'accueil (420x480px recommandé)

## Dossier `projects/`
- `fortigate-lab.jpg` - Capture du lab FortiGate/GNS3 (600x400px recommandé)
- `chaincacao.jpg` - Capture du projet ChainCacao (600x400px recommandé)
- `spectre.jpg` - Capture du projet SPECTRE (600x400px recommandé)
- `portfolio.jpg` - Capture de ce portfolio (600x400px recommandé)

## Instructions
1. Placez vos images dans les dossiers correspondants
2. Assurez-vous que les noms correspondent exactement
3. Optimisez les images (compresser avant upload)
4. Dans les fichiers HTML, remplacez les URLs `picsum.photos` par `assets/images/...`
